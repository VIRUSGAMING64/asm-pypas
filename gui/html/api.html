api reference
api
Base URL
http://localhost:8000
POST /api/run
Body JSON: {"name": "file.asm", "code": "..."}
Ejecuta el codigo y devuelve la salida.
Errores posibles: 400, 404, 500.
POST /api/save
Body JSON: {"name": "file.asm", "code": "..."}
Guarda el archivo en memoria y disco.
Respuesta: {"status":"ok"}.
POST /api/getcode?name=<file>
Query param: name
Devuelve un archivo puntual.
Respuesta: {"status":"ok","code":"..."}.
GET /api/getcodes?name=<file>
Query param: name
Retorna estado y contenido del archivo.
Respuesta: {"status":"ok","code":"..."}.
GET /api/initcodes
No params
Lista nombres de codigos disponibles.
Respuesta: {"status":"ok","names":[...]}
GET /api/newcode?name=<file>
Query param: name
Crea una entrada de codigo vacia.
Respuesta: {"status":"ok","code":""}.
GET /api/delcurr?name=<file>
Query param: name
Elimina el archivo actual.
Respuesta: {"status":"ok"}.
